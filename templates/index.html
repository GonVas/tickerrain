    About

    TickerRain is an open-source webapp that stores and analisys reddit posts in a transparent and semi-interactive manner.

    First it starts a Python process that asynchronous gets posts and comments from subreddits like wallstreetbets, options, thetagang, stockmarket, stocks and some more. The posts are stord in the DB and then another process periodacally processes the posts.

    Processing consists of using Pandas to compute the aggregated mentions, score (based on votes) and sentiment of each stock ticker. Sentiment is computed using Spacy library, utilizing VADER. The sentiment analysis of the last post is shown at the orange box before being added to the final plots.

    Everything is Open-source and the code for both the server and the client can be seen here.

    Plots are updated every 120 seconds.
